## ◈ Concepts Summary

### CI/CD
CI/CD means automatically testing and deploying code whenever changes are made. It helps in faster and reliable development.

### Docker
Docker is used to package applications so they run the same on every system without any issue.

### Cloud (AWS)
Cloud computing allows us to use servers and services online instead of setting up physical machines.
